Adding and Editing Description

﻿Adding a description to your survey can help provide context and clarify any instructions or questions. It can also help respondents understand the purpose of the survey and why their feedback is valuable. After you create your survey and name it, you can start to add a description:


1. Click the area in Add a description to your questionnaire and type in your description in the text box that appears. You can format the text using the formatting toolbar above the text box.






2. To edit an existing description, simply click on the description you want to edit and make your changes.
